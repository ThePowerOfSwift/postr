GET /posts - return a list of posts and associated metadata
POST /posts - create a new post
GET /posts/:id - return an individual post with associated comments
PUT /posts/:id/upvote - upvote a post, notice we use the post ID in the URL
POST /posts/:id/comments - add a new comment to a post by ID
PUT /posts/:id/comments/:id/upvote - upvote a comment



GET /events - return a list of events and associated metadata
POST /events - create a new event
GET /events/:id - return an individual event with associated posters
POST /events/:id/posters - add a new poster to an event by ID
PUT /events/:id/posters/:id/upvote - upvote a poster
